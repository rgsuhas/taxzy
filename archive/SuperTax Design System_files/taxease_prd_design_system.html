01 — Product

Overview

TaxEase AI is a conversational, AI-powered tax filing platform designed to make India's tax system accessible to every citizen — not just chartered accountants.

One-line pitch

"We are building ChatGPT for taxes — an AI-powered platform that helps ordinary people understand, file, and benefit from taxes in the simplest way possible."

Problem Statement

India has ~82 million taxpayers, yet the majority find tax filing stressful, confusing, and opaque. Current systems are built for chartered accountants — not for a 22-year-old freelancer, a salaried employee, or a gig worker.

What's broken today
  • Dense jargon (TDS, 80C, Form 16, ITR)
  • Form-heavy, non-conversational UX
  • Zero refund transparency
  • No explanation of where taxes go
  • High dependency on CAs for simple filings
What we solve
  • Conversational AI guides every step
  • Jargon explained in plain English/Hinglish
  • Real-time refund tracking and marketplace
  • Tax usage visualization builds trust
  • Zero-to-filed in under 15 minutes

03 — Product

Core Features

Seven modules, each addressing a specific gap in today's tax filing experience. Priorities mapped to MoSCoW.

AI Conversational Tax Assistant
Step-by-step guided filing via natural language. Supports English and Hinglish. Understands context (salaried vs freelancer), explains TDS, 80C, HRA, capital gains. Handles follow-up questions. Uses Claude API with system prompt tuned for Indian tax context. Falls back gracefully when uncertain.
Must have
Tax Visualization Dashboard
Real-time cards showing gross income, total TDS deducted, eligible deductions, taxable income, net tax payable, and expected refund. Donut chart for income breakdown. Progress bar for refund status. Recharts-powered with animated transitions as data updates from AI conversation.
Must have
Jargon Simplifier
Inline glossary triggered by underlined terms. Tooltip explains: "TDS = Tax Deducted at Source. Think of it as your employer paying your taxes in advance on your behalf." Covers 40+ terms. AI-powered expansion for unlisted terms.
Must have
Refund Tracker
Timeline view: Filed → Verified → Processing → Refund Initiated → Credited. Estimated date shown. Push notification simulation for demo. Connects to dashboard refund card. Color-coded progress states.
Must have
Refund Marketplace ✦ WOW feature
Time Value of Money concept applied to tax refunds. User with ₹30,000 expected refund can choose ₹25,000 Amazon/Flipkart/Swiggy vouchers today. Platform receives full refund later. Shows conversion rate and instant vs. wait comparison. Framed as demo concept for hackathon.
Must have
Tax Usage Visibility Map
After filing, animated breakdown of where user's tax rupees go: Roads (12%), Railways (8%), Healthcare (15%), Education (18%), Defence (13%), Digital India (6%), Other (28%). Personalized: "Your ₹22,000 funded 3 months of a government school teacher's salary." Drives emotional connection.
Should have
Guided Tax Filing Flow
For users who prefer structured flow over chat: step-by-step wizard with progress indicator. Segments: Personal Info → Income Sources → Deductions → Review → File. Each screen has AI helper button. Pre-fills from chat context if available.

12 — Design

Motion Guidelines

Animations should feel purposeful and joyful — never flashy. Every animation should guide attention, provide feedback, or celebrate a user action.

Motion philosophy
Use Framer Motion in production React components. CSS transitions for micro-interactions in static HTML. Prefer spring curves for delight moments, ease-in-out for navigation.
Interactive Demo — Hover the card

Spring bounce on hover · cubic-bezier(0.34, 1.56, 0.64, 1) · 200ms

Key Animated Moments
Dashboard number countup
When AI extracts income/TDS from chat, dashboard cards animate 0 → final value over 800ms with ease-out. Confirms data was captured and creates delight.
Refund redemption confetti
On clicking "Redeem", canvas-confetti fires in brand orange and teal. Card flips to show voucher code. Duration: 1200ms.
Chat bubble entrance
Each new message slides in from bottom with opacity 0→1 over 200ms. AI typing indicator shows 3-dot pulse before response arrives. Framer Motion layout animation for smooth height changes.
Tax usage map reveal
India map SVG segments animate in staggered sequence (100ms delay each). Percentage labels count up. Total takes 1.5s for full reveal — slow it down for emotional impact.
